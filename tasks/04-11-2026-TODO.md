## Review — Dialogue (Phase 12)

All 90 server tests still passing. `cai dialogue` registered and help verified.

Key implementation notes:
- `_SpeakFileHandler` mirrors `_TextFileHandler` from `watch.py` with two additions:
  a shared `threading.Lock` acquired before every TTS call, and a `threading.Event`
  checked before/after the lock to skip work after shutdown.
- Listener loop (`_listener_loop`) checks `shutdown` between recordings and after
  acquiring the lock — prevents a stale STT call after Ctrl+C.
- `shutdown.set()` → observer.stop() → observer.join() → listener_thread.join(timeout=5)
  join order ensures clean teardown with no deadlocks.
- Listener thread is daemon=True so it cannot block process exit if join times out.
